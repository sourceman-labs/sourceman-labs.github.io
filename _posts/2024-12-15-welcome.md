# Welcome to Sourceman

This is the first post on the new site, built with .NET Razor SSG and deployed to GitHub Pages.

## Features

- **Simple workflow**: Write posts in markdown, push to main, done
- **Git-based**: Version control for all content
- **Preview locally**: `dotnet run` and check before publishing
- **No CMS needed**: Everything lives in the repository
- **Pre-rendered**: Syntax highlighting at build time, zero runtime JS for code blocks

## Code Example

Here's a simple TypeScript function:

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## What's Next?

New posts go in the `_posts/` directory as markdown files with YAML frontmatter. Push to `main` and GitHub Actions handles the rest.

Happy reading!
